Сцена 2D игры на Unity (Top Down shooter)

В проекте реализована механика ходьбы по джойстику, стрельбы по кнопке, инвентарь (подбор и сброс предмета), полосы здоровья для героя и врагов. При нажатии на инвентарь можно его открыть/закрыть. Добавлены звуки мобов и ходьбы, а также простые эффекты при нанесении урона. 
Ллокация создана с помощью Tilemap 2D.
