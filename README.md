Сцена 2D игры на Unity (Top Down shooter)

В проекте реализована механика ходьбы по джойстику, стрельбы по кнопке, инвентарь (подбор и сброс предмета), полосы здоровья для героя и врагов. При уничтожении моба выпадает предмет. При нажатии на инвентарь можно его открыть/закрыть.  Добавлены звуки мобов и ходьбы, а также простые эффекты при нанесении урона. 
Локация создана с помощью Tilemap 2D.

Также я являюсь композитором  (проект "Анимур Миядзаки") https://music.yandex.uz/artist/9232452. Оригинальный саундтрек написан мной.
